El blog del blog (06/09/2024)

Existe un ecosistema enorme para crear blogs o web estáticas simples, por ejemplo, antes había usado Astro para hacer mi portafolio de desarrollador, pero se terminó convirtiendo en una copia de la información que ya tenía en LinkedIn sumado a un par de proyectos que ya había realizado. Otra alternativa que descubrí mientras buscaba una alternativa más simple fue Jekyll, pero no me motivaba mucho tener que instalar Ruby, RubyGems, GCC and Make para hacer mi portafolio. Buscando más alternativas, dí con Mono.css, un framework css bastante simple que aplica estilos de forma automática a los archivos html, bastan con incluir la siguiente línea al header del index.html 

<link rel="stylesheet" href="https://unpkg.com/mono.css@latest/dist/mono.min.css"">
Luego hice el deploy en GitHub pages.
